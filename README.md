# student-card_report_system
This project is a Student Card Report System developed in C++. It allows for the management and generation of student reports, enabling users to easily create, view, and modify student records.

**Features:**

**Student Management:** Add, edit, and delete student records.

**Report Generation:** Generate detailed reports for individual students.

**Data Validation:** Ensures that all data entered is accurate and conforms to the required formats.

**User-Friendly Interface:** Simple command-line interface for easy navigation and usage.

**Programing language Used:**
* C++
* Standard Template Library (STL) for data management
* Basic file handling for data persistence
